ChatterPals 개인정보 처리방침
최종 업데이트일: 2025년 10월 24일
제품: ChatterPals Chrome 확장 프로그램 개발자: ChatterPals Team

1. 개요

ChatterPals(이하 “본 확장 프로그램”)은 사용자가 웹사이트를 탐색하면서 영어를 자연스럽게 학습할 수 있도록 돕는 Chrome 확장 프로그램입니다. 본 프로그램은 웹페이지 일부 텍스트를 가리고, 사용자가 퀴즈 형태로 문장을 완성하거나 영어 표현을 익힐 수 있는 기능을 제공합니다.

본 개인정보 처리방침은 ChatterPals가 사용자 데이터를 어떻게 수집, 이용, 저장 및 공유하는지를 명확히 안내하기 위해 작성되었습니다.

2. 수집하는 데이터

본 확장 프로그램은 다음과 같은 데이터를 처리할 수 있습니다:

데이터 종류 설명 저장 위치
학습 관련 설정 사용자가 선택한 학습 모드, 난이도, 언어 설정 등 로컬 브라우저 저장소 (Chrome Storage API)
퀴즈 진행 정보 사용자가 학습 중 푼 문제, 정답 여부 등 로컬 저장소
확장 기능 사용 로그 (선택) 오류 분석 및 기능 개선을 위한 최소한의 비식별 데이터 서버(chatterpals.onrender.com) 또는 로컬 환경

※ 본 확장 프로그램은 개인 식별 정보(PII), 계정 정보, 비밀번호, 쿠키 데이터, 웹사이트 방문 내역을 수집하지 않습니다.

3. 데이터 이용 목적

  1. 사용자 학습 경험 향상 (맞춤형 퀴즈, 난이도 조정 등)
  2. 오류 및 성능 개선을 위한 기술적 분석
  3. 로컬 학습 기록 저장을 통한 사용자 편의 제공

사용자의 동의 없이 제3자에게 데이터를 판매하거나 광고 목적으로 활용하지 않습니다.

4. 데이터 공유 및 제3자 제공

ChatterPals는 다음의 경우를 제외하고 사용자 데이터를 외부에 제공하지 않습니다:

  • 법적 요청(수사기관, 법원 등)에 따른 의무적인 제출
  • 서비스 운영을 위한 기술적 처리(예: onrender 서버에서 일시적 API 응답 처리)

이 경우에도 데이터는 익명화된 형태로만 처리됩니다.

5. 데이터 보관 및 삭제

  • 사용자의 학습 데이터 및 설정은 로컬 브라우저 저장소에만 저장되며, 사용자가 브라우저 데이터를 삭제하거나 확장 프로그램을 제거하면 함께 삭제됩니다.
  • 서버 통신 데이터(예: 문장 분석 요청)는 일시적으로 처리 후 즉시 폐기됩니다.

6. 보안 조치

  • HTTPS 기반 암호화 통신
  • 로컬 저장소 내 데이터 접근 제한
  • 서버 요청 시 인증 토큰 검증

7. 사용자의 권리

  • 브라우저 설정에서 ChatterPals 데이터 삭제
  • Chrome 확장 프로그램 제거를 통한 모든 로컬 데이터 삭제
  • 개발자에게 문의하여 데이터 관련 요청 (삭제, 수정, 확인 등)

문의: chatterpals.team@gmail.com

8. 변경사항 고지

본 개인정보 처리방침은 필요 시 수정될 수 있으며, 변경 시 Chrome 웹 스토어 페이지 및 확장 프로그램 내 공지사항을 통해 안내됩니다.

요약

  • ChatterPals는 사용자의 개인 정보를 수집하거나 추적하지 않습니다.
  • 모든 학습 데이터는 사용자의 로컬 환경에서만 처리됩니다.
  • 외부 전송은 오직 AI 학습 피드백 요청(API 통신)에 한정됩니다.
